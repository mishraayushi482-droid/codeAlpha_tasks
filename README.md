# Classic Hangman Game
Welcome to my **Hangman Game**! This is a Classic, Text-Based word Guessing game bulit using Python.
The Computer randomly selects a Secret Word from a Predefined List, and the Player must Guess it letter by letter before running out of Attempts.
# Features
- **Random Selection:**
  Automatically selects a word from a list of fun categories (Apple, Banana, Laptop, Sunset, Guitar).
- **Live Tracking:**
  Displays your Current progress with Underscores('_') and Keeps Track of your Remaining Attempts.
- **Case-Insensitive:**
  Accepts Both Uppercase and Lowercase inputs seamlessly.
# The Word List
The Game Features a Curated List of Words to Guess:
* Apple
* Banana
* Laptop
* Sunset
* Guitar
# How to Play:
1. Open your Terminal, Command Prompt, or Python IDLE.
2. Run The Script:
   ```bash
   python "Hangman Game.py"
3. You will Start with 6 Attempts.
4. Type a Letter and hit Enter to make a guess.
5. Guess the full word Correctly to Win! if your Attempts drops to 0, it's Game Over.
